🛠️ Getting Ready

To bring Kriti to life on your computer, we just need to gather a few magical ingredients!

# Install the main cozy terminal packages!
pip install blessed requests

# If you want Kriti to talk and listen (Voice Layer):
pip install pyttsx3 pyaudio SpeechRecognition
A tiny note for Mac users: You might need to run brew install portaudio before installing pyaudio! On Windows, it should work like an absolute charm right out of the box. 🌟

🚀 How to Play

Ready to start your adventure? Just open up your terminal, make sure Ollama is happily running, and type:

# Start Ollama first!
OLLAMA_ORIGINS=* ollama serve

# Then, in a new terminal window, wake up Kriti:
python3 kriti.py
